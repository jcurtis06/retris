### Info
To the untrained eye, the way I define tetriminos looks very random

```js
{
  // S
  shape: [
    { x: 1, y: 0 },
    { x: 2, y: 0 },
    { x: 0, y: 1 },
    { x: 1, y: 1 },
  ],
  width: 3,
  height: 2,
},
{
  // Z
  shape: [
    { x: 0, y: 0 },
    { x: 1, y: 0 },
    { x: 1, y: 1 },
    { x: 2, y: 1 },
  ],
  width: 3,
  height: 2,
},
```
